# Domain Driven Design

### Caso de uso:
- Para estudar DDD na prática, selecionei um caso de uso de um setor logístico.
  
### Stack utilizada:
- **Typescript**
- **Node 22.15.0**
- **API**: Express
- **Banco** de dados: MySQL
- **ORM**: Prisma
- **Testes**: Vitest

### NPM scripts
```bash
npm run prisma:generate # inicia o schema do prisma

npm run dev # hot reload da api

npm run test # testes automatizados
```
